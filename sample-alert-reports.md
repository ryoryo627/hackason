## A-1: バイタル低下トレンド（HIGH）

**検知条件**: SpO2/BP 等が連続2回以上低下

### 過去7日間の報告履歴（3件セット）

**報告①** 2/4（火）訪問看護師 山本

```
訪看 山本です。田中太郎さん本日訪問。
バイタル: SpO2 96%（RA）、BP 138/82、HR 78、BT 36.4℃
SpO2は先週の98%から少し下がっていますが、自覚症状なし。
労作時の息切れも特になく、経過観察とします。
内服は処方通り継続中。
```

**報告②** 2/6（木）訪問看護師 山本

```
訪看 山本です。田中太郎さん訪問。
バイタル: SpO2 94%（RA）、BP 142/86、HR 82、BT 36.6℃
前回96%から更に低下しています。安静時でもやや息苦しさの訴えあり。
「階段を上がると前より疲れる」との発言。
排痰は少量、色は白色。食事摂取量はやや減少（8割程度）。
次回訪問時も低下傾向が続くようであれば主治医に報告予定。
```

**報告③（最新）** 2/8（土）訪問看護師 佐々木

```
訪看 佐々木です。山本の代行で田中太郎さん訪問しました。
バイタル: SpO2 92%（RA）、BP 148/90、HR 86、BT 36.8℃
SpO2が3回連続で低下しています（98→96→94→92）。
安静時でも軽度の呼吸困難あり、会話中に息継ぎが目立ちます。
肩呼吸は見られませんが、呼吸数22回/分とやや促迫。
食事量は6割程度に低下。下肢浮腫なし。
主治医への早期報告が必要と考えます。
```

**→ 想定されるアラート出力**:
- pattern_id: A-1
- severity: HIGH
- title: SpO2連続低下トレンド
- evidence: SpO2が98%→96%→94%→92%と4回連続低下、安静時呼吸困難出現
- recommendations: 主治医への早期相談、胸部X線・採血の検討

## A-2: 複合Bio悪化（HIGH）

**検知条件**: 2項目以上のBio指標が同時悪化

### 報告（単一報告で複数指標悪化）

**報告** 2/9（日）訪問看護師 山本

```
訪看 山本です。佐藤花子さん本日訪問。
バイタル: SpO2 93%（前回96%）、BP 158/94（前回132/78）、
HR 98（前回72）、BT 37.4℃（前回36.3℃）、体重 +1.8kg（1週間）

全体的にバイタルが悪化しています。
SpO2低下に加え、血圧が著明に上昇、頻脈も出現。
微熱あり、CRP上昇の可能性。
体重増加が顕著で、両下肢に圧痕性浮腫（+2）を認めます。
食欲低下あり、昼食は3割残し。水分摂取も減少気味。
呼吸音：右下肺野で湿性ラ音聴取。
「身体がだるくて起き上がるのがつらい」との訴え。

ADL低下あり、入浴は中止し清拭対応としました。
内服のフロセミド20mgは継続中ですが、効果不十分の印象です。
利尿剤の増量について主治医への相談をお願いします。
```

**→ 想定されるアラート出力**:
- pattern_id: A-2
- severity: HIGH
- title: 複合Bio指標悪化（心不全増悪疑い）
- evidence: SpO2低下(96→93%)、血圧上昇(132→158)、頻脈出現(72→98)、体重増加(+1.8kg/週)、下肢浮腫増悪、湿性ラ音
- recommendations: 診察・血液検査（BNP含む）・胸部X線の実施、利尿剤調整の検討

## A-4: Bio+Social複合（MEDIUM）

**検知条件**: 服薬低下 + 介護負担増加

**報告** 2/7（金）薬剤師 木村

```
薬剤師 木村です。渡辺よし子さんの居宅訪問薬剤管理。
お薬カレンダーを確認したところ、今週分のアムロジピン5mg と
メトホルミン500mg が3日分残っていました。
「飲み忘れちゃったみたい」とのことですが、
ご主人（主介護者）に確認すると「最近声かけする余裕がなくて…」
とのこと。ご主人自身も腰痛が悪化しており、
整形外科の通院が増えているようです。
残薬調整を行い、一包化の提案をしましたが、
ご主人が「薬局まで取りに行く元気がない」と消極的でした。
次回は訪看さんと情報共有し、服薬管理の方法を再検討したいです。
```

### 報告②（看護師からの補足）

**報告** 2/9（日）訪問看護師 加藤

```
訪看 加藤です。渡辺よし子さん訪問。
バイタル: BP 162/96（前回138/82）、HbA1cは先月の7.2%から
今月7.8%に上昇（主治医からの検査結果共有）。

薬剤師の木村さんから服薬アドヒアランス低下の報告を受け確認。
ご本人は「主人が忙しくて薬を出してもらえない日がある」と話す。
ご主人（82歳）は腰椎圧迫骨折で歩行が不安定になっており、
家事全般が困難な状況。洗い物がシンクに溜まっていました。
ゴミ出しもできていない日があるとのこと。
ヘルパーの利用を提案しましたが「他人に家に入られたくない」と拒否。
介護者の共倒れリスクが高まっている印象です。
服薬管理方法の見直しと、介護サービスの再調整が急務と考えます。
```

**→ 想定されるアラート出力**:

- severity: MEDIUM
- title: 服薬アドヒアランス低下＋介護負担増加
- evidence: 週3日分の残薬、血圧上昇(138→162)、HbA1c悪化(7.2→7.8%)、主介護者の身体機能低下（腰椎圧迫骨折）、家事困難
- recommendations: 服薬管理方法の見直し（一包化・配薬ボックス等）、介護サービス増量の検討、地域包括支援センターとの連携

## 使い方

1. 上記の `raw_text` をSlackの患者チャンネル（例: `#pt-田中太郎`）に投稿
2. IntakeAgent が BPS 構造化を実行
3. 朝8時の `morning-scan` cron で AlertAgent が過去7日間の報告を分析
4. 該当パターンが検知されると、`#oncall-night` と患者チャンネルにアラート投稿

### テスト時の注意
- A-1 は**履歴との比較**が必要なため、報告を数日分に分けて時系列で投稿する
- A-5 は**複数職種**からの報告が根拠となるため、異なるreporterで投稿する
- A-6 は**家族からの報告**であることが検知条件の重要な要素
